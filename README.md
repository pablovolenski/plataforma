# Plataforma

A starter repository for a publishing platform where access is granted only to predefined user accounts. Authenticated users can write articles and publish them to a public wall.

## Core rules

- Accounts are curated by administrators and are not self-service.
- Authors can create posts in one of three categories: `opinion`, `news`, or `event`.
- Published posts appear on a public wall visible to everyone.

## Current prototype

This first version is a no-build frontend prototype built with plain HTML, CSS, and JavaScript so it can be opened immediately without installing dependencies.

## Suggested next steps

1. Add a real authentication layer backed by a server and database.
2. Persist predefined users and articles.
3. Add moderation, drafts, and approval flows if needed.

## Run locally

Open `src/index.html` in a browser.
